# RW 配置 D 最適化 — 実装計画

## 1. 問題の概要

`position_transform.py` では 6 配置の変換行列 T_i をあらかじめ手動で決めていた。
ここでは **D 最適計画の基準** を用いて、伝達行列同定に最も適した RW 配置（位置・スピン軸方向）を自動決定する。

---

## 2. 物理モデルと U_mat の構造

### 変換行列（`make_T` の再利用）

位置 r、スピン軸方向 d（単位ベクトル）に対して：

```
T(r, d) = [ R          0 ]   (6×6)
           [ S(r)@R    R ]
```

ここで R = rotation_to_align(d)（ローカル x 軸を d に合わせる回転行列）。

### U_mat = [T_1 u | T_2 u | ... | T_6 u] における u の扱い

同定では以下の線形系を解く：

```
Y_mat = G @ U_mat
U_mat[:,i] = T_i @ u(ω)    ← u(ω) は実測前には未知
```

`u(ω)` は RW 本体の物性（質量不釣り合い等）から決まる力学ベクトルであり、配置とは独立に存在する。
**測定前には u の方向は不明である。**

#### ベースライン仮定（静的不釣り合い優位モデル）

RW の主要外乱は **スピン軸方向の力（静的不釣り合い成分）** であるため、
ローカル座標での u を次のように近似する：

```
u_local ≈ F · [1, 0, 0, 0, 0, 0]^T    （スピン軸 = ローカル x 軸）
```

この仮定のもとで：

```
T(r_i, d_i) @ u_local = F · [ d_i        ]
                              [ r_i × d_i ]
```

U_mat の各列がそれぞれの配置のランチベクトル（wrench vector）になる。
F はスカラー倍にすぎず `log|det|` には影響しないため、配置最適化から除外できる。

#### u が未知であることの影響

- u の **方向が周波数によらず一定**（振幅だけ変化 → PSD変化）であれば、上記仮定の範囲で最適化は成立する。
- 高次高調波など u の **方向が変わる**場合は、複数の u_n を想定して以下のような周波数平均基準に拡張する：

```
maximize  Σ_n  w_n · log |det(U_mat(u_n))|
          w_n = PSD(ω_n) / Σ PSD  （PSD重み）
```

まずはベースライン仮定で実装し、測定後に u_n を代入して再評価・再最適化する。

---

## 3. D 最適基準の定式化

### D 最適目的関数

情報行列 M = U_mat @ U_mat^T (6×6) として：

```
maximize  f = log |det(U_mat)|
```

U_mat が 6×6 のとき det(M) = det(U_mat)^2 なので `log|det(U_mat)|` の最大化と等価。
これが参考記事の「det(X^T X) を最大化する」D 最適基準に対応する。

## 4. 設計変数

| 変数 | 内容 | 次元 |
|------|------|------|
| `r_1 = (0,0,0)` | 位置 1（固定） | — |
| `r_2 = (1,0,0)` | 位置 2（固定） | — |
| `r_3 = (x,y,z)` | 位置 3（自由） | 3 |

| `d_21, d_22` | 位置 2 の RW1, RW2 スピン軸 | 2×2 = 4 |
| `d_31, d_32` | 位置 3 の RW1, RW2 スピン軸 | 2×2 = 4 |

**合計 15 自由変数**

### 単位ベクトルの球面パラメータ化

|d|=1 を制約として陽に扱わず、角度変数で自動的に満たす：

```
d(θ, φ) = (sin θ cos φ,  sin θ sin φ,  cos θ)
```

θ ∈ [0, π]、φ ∈ [-π/2, π/2] とすることで **d_x = sin θ cos φ ≥ 0** を保証。
→ 6 つのスピン軸を (θ_k, φ_k) の 12 変数で表現。

### スピン方向の符号不定性について

d と -d はランチベクトルの符号を反転させるだけ（= U_mat の 1 列の符号が変わるだけ）なので、
`|det(U_mat)|` に対して全く同等である。
球面パラメータ化で **φ ∈ [-π/2, π/2]（= d_x ≥ 0）** に制限することで
各スピン軸ごとの 2 重解を一意に除去する。

### r と d のスケールについて

- **d**: |d|=1 に正規化済み → 力ブロック（U_mat の上 3 行）の各列の大きさは常に 1。スケール問題なし。
- **r**: トルクブロック（U_mat の下 3 行）の各列 = r × d の大きさは |r|sin(angle)。
  r の大きさが直接トルクブロックの値を決める。
  **r の物理的スケール（衛星サイズ [m]）は設計制約として陽に設定すること**（次節参照）。
  位置の制約を適切に設ければ、数値的スケール問題は実用上問題にならない。

## 9. 今後の検討事項（実装・議論を踏まえた課題）

### 9-1. 位置の canonical form 再設計（優先度：高）

**現状の問題：**
`R1 = [0, 0, 0]` に固定しているため、R1 に取り付けた RW2 本のトルク寄与が構造的にゼロになる。

```
r = [0,0,0]  →  r × d = 0  →  U_mat のトルク行（下3行）に寄与なし
```

これは座標原点の物理的意味（重心かどうか）とは無関係で、**原点に位置を置くこと自体が退化**を招く。

**検討すべき canonical form：**

座標系の任意性（平行移動3 + 回転3 = 6 DOF）を以下の拘束で除去する：

| 拘束 | 除去する自由度 |
|------|--------------|
| (r1 + r2 + r3) / 3 = 0 （重心を原点） | 平行移動 3 DOF |
| r1 = (a, 0, 0)，a > 0 （r1 を x 軸上） | 回転 2 DOF |
| r2_z = 0，r2_y ≥ 0 （r2 を xy 平面内） | 回転 1 DOF |

これで残る位置自由変数は：

```
r1 = (a, 0, 0)          ← 1変数
r2 = (b, c, 0), c ≥ 0   ← 2変数
r3 = -(r1 + r2)         ← 自動決定
```

合計 3 変数（現在と同じ数）で、全 RW が原点から離れた退化しない配置になる。

---

### 9-2. det(U_mat) 基準の座標系依存性（優先度：中）

**問題の本質：**

`log|det(U_mat)|` は座標変換（特に平行移動）に対して不変ではない。
全位置を同じベクトル δ だけ平行移動すると：

```
U_mat[:,i]_new = [d_i;  (r_i + δ) × d_i]  ≠  U_mat[:,i]_old
```

物理的な情報量は同じだが、**U_mat の最小特異値が変わるためノイズ増幅率も変わる**。

**本来あるべき基準：**

ノイズ共分散 Σ_N で正規化した D 最適基準：

```
maximize  log |det(Σ_N^{-1/2} @ U_mat)|
```

これは座標変換に対して不変になる。

**現在のノイズモデル（周波数域 SNR 指定→逆 FFT）でのΣ_N の推定：**

高調波周波数 ω_n での出力ノイズ共分散は：

```
Σ_N(ω_n) ≈ (1/SNR) × diag(|Y_1(ω_n)|², ..., |Y_6(ω_n)|²)
```

力チャネルとモーメントチャネルで次元・大きさが異なるため、対角成分は均一でない点に注意。

**当面の対処：**
ノイズ力学挙動が不明な段階では、U_mat の力ブロック（上3行）とトルクブロック（下3行）を特性長さ L（衛星代表寸法 [m]）でスケーリングする：

```
U_mat_scaled = diag(1, 1, 1, 1/L, 1/L, 1/L) @ U_mat
maximize  log|det(U_mat_scaled)|
```

これにより力とモーメントの単位を揃え、det が力/トルクのスケール比に依存しなくなる。

### 9-3. 同定精度の周波数依存性（観察事項の整理）

シミュレーション結果から見られた傾向：

| 現象 | 原因 |
|------|------|
| output x 行の推定誤差が大きい | R1=[0,0,0] 配置でトルク行への寄与がゼロ → U_mat のモーメント行が R2, R3 の4列だけに依存 |
| 100 Hz 超でずれ始める成分がある | 高調波次数が上がるにつれ信号強度が低下（~1/n²）し、実効 SNR が悪化 |
| 120 Hz まで正確な成分がある | G の絶対値が大きい（対角・主クロストーク）成分は SNR に余裕があるため |
| cond(U_mat) の増大 | 特定の高調波で U_mat がほぼ退化方向を持ち、ノイズが G_hat に増幅されて寄与 |

**対策の方向：**
- 9-1 の canonical form を採用してトルク行の独立性を改善
- 9-2 のスケーリングで力/トルクブロックのバランスを調整
- 高次高調波に対しては PSD 重み付き基準（セクション 2 の拡張案）を検討

### 9-4. u_local の方向と D 最適化の関係（確認済み事項）

- `det(U_mat) = |u|^6 × det([T_1@û | ... | T_6@û])` という因数分解が成立する
- **振幅 |u|（= F）は det に定数寄与するだけで最適配置に影響しない** → 除外可
- **方向 û は各 T_i から取り出す列方向を決めるため無視できない**
- ベースライン仮定（û = [1,0,0,0,0,0]^T）は T_i の第1列を選ぶことに相当し、現実の RW 外乱に対して合理的な近似
