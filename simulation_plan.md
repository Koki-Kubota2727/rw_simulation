# 6×6伝達行列同定アルゴリズム 検証シミュレーション計画

## 目的

RWの取り付け位置を複数変えることで6×6伝達行列 G(ω) を同定するアルゴリズムが
数値的に成立するかを、Pythonシミュレーションで検証する。

---

## 信号の次元について（重要）

### 入力と出力の関係

RW回転数 Ω に対して、入力・出力はいずれも**周波数の関数**として与えられる。

- U_rw(ω) ∈ C^6: 入力信号 [Fx, Fy, Fz, Mx, My, Mz] の Fourier 変換
  → 各成分は周波数 ω における複素数値（振幅・位相を含む）
- Y(ω) ∈ C^6: 出力信号 [x1, x2, x3, θx, θy, θz] の Fourier 変換

各周波数 ω において：

  Y(ω) = G(ω) * U(ω)   ← 6個の方程式、36個の未知数

1回の計測（1配置）では**不定**（underdetermined）。

---

## アルゴリズムの概要

### 取り付け配置の設定

2か所の取り付け位置 r_1, r_2 を設け、各位置でRWを3方向（x・y・z軸向き）に
直交して設置する。合計 N = 2 × 3 = 6 配置。

取り付け位置の例：

  r_1 = [0.10, 0.05, 0.00] m
  r_2 = [-0.08, 0.00, 0.12] m

向き（RWスピン軸方向）の例：

  e_x = [1, 0, 0]^T   → R_x = I
  e_y = [0, 1, 0]^T   → R_y = [[0, -1, 0], [1, 0, 0], [0, 0, 1]]
  e_z = [0, 0, 1]^T   → R_z = [[0, 0, -1], [0, 1, 0], [1, 0, 0]]

（R_k は RW局所座標系から機体座標系への回転行列）

### 変換行列 T_{j,k}

位置 r_j にRWを向き R_k で取り付けたときの実効入力変換：

  U_{j,k}(ω) = T_{j,k} * U_rw(ω)

  T_{j,k} = [ R_k             0   ]
             [ S(r_j) * R_k   R_k ]

- 上段: 力の回転変換 R_k
- 下段: 位置によるモーメントアーム S(r_j)*R_k と、モーメント直接成分 R_k

S(r) は歪対称行列：

  S(r) = [  0   -rz   ry ]
         [  rz   0   -rx ]
         [ -ry   rx   0  ]

### 各周波数ごとに独立に同定

6配置の計測をスタック：

  Y_mat(ω) = [ Y_{1,x}(ω) | Y_{1,y}(ω) | Y_{1,z}(ω) | Y_{2,x}(ω) | Y_{2,y}(ω) | Y_{2,z}(ω) ]   (6×6)
  U_mat(ω) = [ U_{1,x}(ω) | U_{1,y}(ω) | U_{1,z}(ω) | U_{2,x}(ω) | U_{2,y}(ω) | U_{2,z}(ω) ]   (6×6)

  Y_mat(ω) = G(ω) * U_mat(ω)

→ U_mat(ω) がフルランク（ランク6）なら：

  G(ω) = Y_mat(ω) * inv(U_mat(ω))   （各 ω で独立に計算）

### ランク条件について

U_mat(ω) の各列は T_{j,k} * U_rw(ω)。ランクは位置 r_j と向き R_k の多様性に依存する。
→ 2か所・3方向の配置が適切であれば U_mat がランク6になるかを検証する。
→ 条件数 cond(U_mat) を評価指標とする。　小さいほど誤算に対してロバストにGを同定できることになる。

---

## 入力信号の設計方針

RW の擾乱は回転数 Ω の高調波として現れる：

  u_rw(t) = Σ_n A_n ⊙ sin(n·Ω·t + φ_n)

ここで A_n ∈ R^6 は高調波次数 n での各成分の振幅ベクトル。

時間領域信号を FFT すると、各高調波周波数 ω_n = n·Ω において
U_rw(ω_n) ∈ C^6 が得られる（全成分が非零）。

### 和積公式による解析値との照合

正弦波入力を用いることで、クロスパワースペクトルの理論値と比較できる：

  sin(ω_a t) · sin(ω_b t) = [cos((ω_a - ω_b)t) - cos((ω_a + ω_b)t)] / 2

→ Syu(ω) のピーク位置（ω = ω_a ± ω_b）と振幅が解析的に求まる
→ 同定結果との一致を確認することで実装の正しさを検証できる

---

## 真のG(ω)の設定方針

### 案1: 対角 + 小さな非対角成分（最初の検証用）

  G_diag の各対角成分: 1自由度ばね-マス-ダンパー系
  H_k(ω) = 1 / (m_k * (jω)^2 + c_k * (jω) + k_k)

  → 解析的な FRF と同定結果を直接比較できる

### 案2: ランダム複素行列（ロバスト性検証用）

  G_true(ω) = ランダム生成した複素行列（各周波数で独立）

  → アルゴリズム自体の数値的成立を確認

## 実装ステップ

### Step 0: 準備
- numpy, scipy, matplotlib をインポート
- パラメータ設定（サンプリング周波数 fs、時間長 T、RW回転数 Ω、高調波次数 n_max）

### Step 1: 真のG(ω)を定義
- 案1の対角+非対角形式で、各周波数点での 6×6 複素行列として計算

### Step 2: 取り付け配置の設定
- r_1, r_2 を設定し、3方向の回転行列 R_x, R_y, R_z を定義
- 6通りの変換行列 T_{j,k} を計算

### Step 3: 入力信号の生成
- 高調波モデルで u_rw(t) を生成（6次元時系列）
- FFT により U_rw(ω) を計算
- 6配置それぞれの実効入力 U_{j,k}(ω) = T_{j,k} * U_rw(ω) を計算

### Step 4: 出力の計算（順問題）
- 各周波数 ω_k で Y_{j,k}(ω_k) = G_true(ω_k) * U_{j,k}(ω_k) を計算
- IFFT で時間領域 y(t) に戻す（必要に応じてノイズを加算）

### Step 5: G(ω)の同定（逆問題）
- 各周波数 ω_k について U_mat(ω_k), Y_mat(ω_k) を 6×6 にスタック
- G_hat(ω_k) = Y_mat * inv(U_mat) を計算
- cond(U_mat) を記録

### Step 6: 検証
- G_hat vs G_true を各成分で比較（振幅・位相のボード線図）
- 誤差: ||G_hat - G_true||_F / ||G_true||_F
- Syu(ω) の和積公式による理論値との比較
- r_1, r_2 の配置を変えたときの cond(U_mat) の変化を確認
